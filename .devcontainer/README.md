# Betaflight DevContainer with OpenCode Integration

This devcontainer is configured with OpenCode AI assistant integration for enhanced development experience.

## Features

- Full Betaflight firmware development environment
- ARM cross-compilation toolchain
- OpenCode AI assistant integration
- VS Code extensions for embedded development

## OpenCode Usage

Once the container is running, you can use OpenCode commands in your terminal:

```bash
# View available OpenCode commands
opencode --help

# Get help with a specific command
opencode help <command>

# Use OpenCode to explain code
opencode explain <file> <line-number>
```

## VS Code Integration

The OpenCode extension is automatically installed in the devcontainer, providing:
- AI-powered code suggestions
- Code explanation capabilities
- Smart debugging assistance
